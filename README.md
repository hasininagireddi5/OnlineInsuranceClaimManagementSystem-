# Online Insurance Claim Management System

The Online Insurance Claim Management System is a web-based application developed to simplify the process of managing insurance policies and claims. This system allows customers to view insurance policies, submit claims, upload documents, and track claim status online. It also helps administrators manage claims, policies, and payments efficiently.

---

## Features

- Customer login and dashboard
- View available insurance policies
- Raise insurance claims
- Upload claim documents
- Track claim status
- Payment information management
- Notifications for claim updates
- Admin dashboard for system monitoring

---

## Technologies Used

- **HTML** – Web page structure  
- **CSS** – User interface design  
- **JavaScript** – Application functionality  
- **Local Storage** – Data storage simulation  
- **SQL Concepts** – Database query demonstration

---

## System Modules

### Customer Module
Allows customers to login, view policies, raise claims, and check claim status.

### Policy Management Module
Manages insurance policies including policy details and premium information.

### Claim Management Module
Handles claim submission, claim processing, and claim status updates.

### Document Management Module
Manages uploading and verification of claim-related documents.

### Payment Management Module
Records payment details after claim approval.

### Notification Module
Sends updates to customers regarding claim status and system notifications.

### Database Management Module
Stores and manages system data including customers, policies, claims, and payments.
